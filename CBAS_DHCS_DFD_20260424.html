▪ FILTER — 6-MONTH DEDUP
#CBAS_S5102_NEWONLY

NOT EXISTS: same member + same service code + prior 6 months

▶ STEP 2 — T1023 MATCH (COLUMNS I / H)
▪ JOIN
#CBAS_T1023_MATCH
For each S5102 member from #NEWONLY:
○ Find T1023 (Face-to-Face) auth within ±90 days
○ Compute: T1023 Receipt → S5102 Decision Date
○ Flag: Within_30_Days_Flag (1 = ≤30 days)

Column I = Count of flags = 1
Column J = Total from Column G
DATEDIFF(day, T1023_Receipt, S5102_Decision) <= 30
▪ JOIN
#CBAS_H2000_MATCH
For each S5102 member (Approved/Denied only):
○ Find T1023 OR H2000 auth within ±90 days
○ Presence flag = member had F2F or in-facility assessment

Column L = Count with assessment present
Column M = Total Approved + Denied
servicecode IN ('T1023','CBAS ASSESS') within ±90 days
