Key Prompt Components

  • Role Assignment: "ACT AS a Senior Data Analyst"
  • Experience Specification: "WITH 20+ years of EXPERIENCE"
  • Task Description: "PROVIDE", "EXAMINE", "EXPLAIN", "INTERPRET"
  • Output Format: "#MERMAID FLOW CHART", "DETAILED report"
  • Focus Areas: "PARAMETERS", "FROM & JOIN clauses", "WHERE CLAUSE", "WINDOW FUNCTIONS"

Specialized Prompt Types

  1. Code Analysis and Review
  2. Code Optimization
  3. Debugging Assistance
  4. Code Conversion (e.g., to Snowflake)
  5. Interactive Visualization Creation
  6. Documentation Generation
  7. Specification Alignment Check

Best Practices

  • Be specific about the role and experience level
  • Clearly define the task and desired output format
  • Provide focus areas or specific aspects to address
  • Use capitalization for emphasis on key instructions
  • Include requests for step-by-step explanations when needed
  • Specify code style preferences (e.g., comment retention, aliasing)
